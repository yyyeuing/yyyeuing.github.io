Simple Sentiment Analysis

数据处理

建立词表：机器不能处理字符串，要将其转换成数字。每个索引用于为每个单词构造一个热向量（one-hot:除了1位，其他所有元素都是0）。由于词汇过多，训练会十分缓慢。可以考虑将一些出现次数少于m次的词进行丢弃或者只选用常出现的n个词汇进行训练。而那些被丢弃的单词可以用来代替

搭建网络

three layers:embedding layer,RNN,linear layer

  • embeddiong: transform our sparse one-hot vector into a dense embedding vector.The embedding layer is simply a single fully connected layer
  • RNN: take in dense vector and the previous hidden state ht-1 –> calculate the next hidden state ht
  • linear layer: takes the final hidden state and feeds it through a fully connected layer, transforming it to the correct output dimension.
